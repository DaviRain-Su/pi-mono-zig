快速开始

Getting started

单二进制，无需安装运行时。

Single binary, no runtime to install.

# 构建
zig build -Doptimize=ReleaseFast

# 交互模式
./zig-out/bin/pi-agent

# 一次性 prompt
./zig-out/bin/pi-agent --prompt "帮我审查 src/main.zig 的内存管理"

# JSON 事件流（机器消费）
./zig-out/bin/pi-agent --json --prompt "..."

# 查看可用模型
./zig-out/bin/pi-agent --list-models

# 续上一会话
./zig-out/bin/pi-agent --continue
# Build
zig build -Doptimize=ReleaseFast

# Interactive mode
./zig-out/bin/pi-agent

# One-shot prompt
./zig-out/bin/pi-agent --prompt "Review memory management in src/main.zig"

# JSON event stream (for machines)
./zig-out/bin/pi-agent --json --prompt "..."

# List available models
./zig-out/bin/pi-agent --list-models

# Continue last session
./zig-out/bin/pi-agent --continue
